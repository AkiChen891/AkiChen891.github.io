Markdown编译器

在 Hexo 中，如果默认的 Markdown 编译器无法正确渲染 LaTeX 公式，可以考虑使用支持 LaTeX 公式的 Markdown 编译器插件。常用的包括：

  • hexo-renderer-pandoc
  • hexo-renderer-markdown-it
  • hexo-filter-mathjax
  • hexo-renderer-markdown-it-plus
