简化十二相同步整流发电机模型推导

引言

十二相同步发电机在定转子分解方面的步骤和三相同步发电机基本一致。如果单独考虑与定子直轴和交轴绕组想独立的零轴绕组，则在考虑ddd、qqq、fff、DDD、QQQ五个绕组的电磁过渡过程及转子机械过渡过程时，会面临高阶模型的求解难题。实际工程中通常对同步电机的数学模型作不同程度的简化。

假定：

  • 忽略定子直轴、交轴绕组的暂态过程，即定子电压方程中取pΦd=pΦq=0p\Phi_d=p\Phi_q=0pΦd​=pΦq​=0

  • 忽略DDD、QQQ和等效交轴稳定绕组，其作用可在转子运动方程补入阻尼项来近似

导出思路

思路如下：

  1. 对于Park方程，在忽略DDD、QQQ绕组（将相应方程及变量删去）后，尚有以下变量：udqfu_{dqf}udqf​、idqfi_{dqf}idqf​、Φdqf\Phi_{dqf}Φdqf​及ω\omegaω、δ\deltaδ、TmT_mTm​。假设ufu_fuf​和TmT_mTm​为已知量（分别为励磁系统及原动机输出量），则有10个未知量。对应有直轴、交轴和励磁3个绕组的电压方程、磁链方程和2个转子运动方程，共计8个方程。若和直轴、交轴网络方程联立，则变量数和方程数平衡，可以求解。
  2. 由Park方程推导三阶实用模型时，应保留定子变量udqu_{dq}udq​和idqi_{dq}idq​，而转子变量ufu_fuf​、ifi_fif​和Φf\Phi_fΦf​分别用EfE_fEf​、EqE_qEq​和Eq′E_q'Eq′​替代，然后再用3个磁链方程消去Φd\Phi_dΦd​、Φq\Phi_qΦq​及ifi_fif​（或EqE_qEq​），从而在最终同步电机模型中保留udqu_{dq}udq​、idqi_{dq}idq​、EfE_fEf​、Eq′E_q'Eq′​及ω\omegaω、δ\deltaδ、TmT_mTm​等变量，其中Eq′E_q'Eq′​、ω\omegaω、δ\deltaδ为状态量，电机方程由3个电压方程和2个转子运动方程组成。当EfE_fEf​和TmT_mTm​已知时（对外表现为励磁系统和原动机系统输入），可联立求解方程。

变量转换及消去

先消去Φd\Phi_dΦd​、Φq\Phi_qΦq​及ifi_fif​（EqE_qEq​）所用表达式，将其用保留变量的函数来表示。

已知直轴磁链方程为：

Φd=−Xdid+Xadif\Phi_d = -X_di_d + X_{ad}i_f Φd​=−Xd​id​+Xad​if​

Φf=−Xadid+Xfif\Phi_f = -X_{ad}i_d + X_{f}i_f Φf​=−Xad​id​+Xf​if​

将上式中的第二个式子等式两侧同乘Xad/XfX_{ad}/X_fXad​/Xf​，有

Eq′=−Xad2Xfid+XadifE_q' = -\frac{X_{ad}^2}{X_f}i_d + X_{ad}i_f Eq′​=−Xf​Xad2​​id​+Xad​if​

因为

Xd′=Xd−Xad2XfX_d' = X_d - \frac{X_{ad}^2}{X_f} Xd′​=Xd​−Xf​Xad2​​

则上上个式子可表示为

Eq′=Φd+Xd′idE_q' = \Phi_d + X_d'i_d Eq′​=Φd​+Xd′​id​

或

Φd=Eq′−Xd′id\Phi_d = E_q' - X_d'i_d Φd​=Eq′​−Xd′​id​

上式即可用来消去Φd\Phi_dΦd​。

由Φd=−Xdid+Xadif\Phi_d = -X_di_d + X_{ad}i_fΦd​=−Xd​id​+Xad​if​和Eq=XadifE_q = X_{ad}i_fEq​=Xad​if​，可得

Φd=Eq−Xdid\Phi_d = E_q - X_di_d Φd​=Eq​−Xd​id​

将Φd=Eq′−Xd′id\Phi_d = E_q' - X_d'i_dΦd​=Eq′​−Xd′​id​代入上式，可得

Eq′−Xd′id=Eq−XdidE_q' - X_d'i_d = E_q - X_di_d Eq′​−Xd′​id​=Eq​−Xd​id​

整理，上式即为

Eq=Eq′+(Xd−Xd′)idE_q = E_q' + (X_d - X_d')i_d Eq​=Eq′​+(Xd​−Xd′​)id​

该式可用于消去ifi_fif​(EqE_qEq​)。该式同时也是忽略DDD、QQQ绕组时用于表征EqE_qEq​和Eq′E_q'Eq′​之间的一个重要关系。

注意：计及DDD、QQQ绕组暂态时，此关系不成立。

qqq轴磁链方程为：

Φq=−Xqiq\Phi_q = -X_qi_q Φq​=−Xq​iq​

该式可直接用来消去Φq\Phi_qΦq​。

至此，消去Φd\Phi_dΦd​、Φq\Phi_qΦq​和ifi_fif​(EqE_qEq​)的表达式均已得到，分别为：

Φd=Eq−Xdid\Phi_d = E_q - X_di_d Φd​=Eq​−Xd​id​

Φq=−Xqiq\Phi_q = -X_qi_q Φq​=−Xq​iq​

导出三阶模型

对原始电压方程，令pΦd=pΦq=0,ω≈1p\Phi_d=p\Phi_q=0, \omega \approx 1pΦd​=pΦq​=0,ω≈1，再将Φd=Eq−Xdid\Phi_d = E_q - X_di_dΦd​=Eq​−Xd​id​和Φq=−Xqiq\Phi_q = -X_qi_qΦq​=−Xq​iq​代入，得

{ud=Xqiq−raiduq=Eq′−Xd′id−raiq \left\{ \begin{aligned} u_d = X_qi_q - r_ai_d \\ u_q = E_q' - X_d'i_d - r_ai_q \end{aligned} \right. {ud​=Xq​iq​−ra​id​uq​=Eq′​−Xd′​id​−ra​iq​​

对于励磁绕组电压方程，将其改写为

将Eq=Eq′+(Xd−Xd′)idE_q = E_q' + (X_d - X_d')i_dEq​=Eq′​+(Xd​−Xd′​)id​代入上式，消去EqE_qEq​，有

上式即为转子励磁绕组的暂态方程。

另对转子运动方程改造如下：

根据

TJdωdt=Tm−Te=Tm−[Eq′iq−(Xd′−Xq)idiq]T_J \frac{d\omega}{dt} = T_m - T_e = T_m - [E_q'i_q - (X_d' - X_q)i_di_q] TJ​dtdω​=Tm​−Te​=Tm​−[Eq′​iq​−(Xd′​−Xq​)id​iq​]

{ud=Xqiq−raiduq=Eq′−Xd′id−raiq \left\{ \begin{aligned} u_d = X_qi_q - r_ai_d \\ u_q = E_q' - X_d'i_d - r_ai_q \end{aligned} \right. {ud​=Xq​iq​−ra​id​uq​=Eq′​−Xd′​id​−ra​iq​​

Td0′pEq′=Ef−Eq′−(Xd−Xd′)idT_{d0}'pE_q' = E_f - E_q' - (X_d - X_d')i_d Td0′​pEq′​=Ef​−Eq′​−(Xd​−Xd′​)id​

推广至十二相同步发电机

对于发电机而言，输入量通常为励磁系统（ufu_fuf​或ifi_fif​）和原动机（TmT_mTm​或ω\omegaω），每个系统的两个输入量均可互相转换。
